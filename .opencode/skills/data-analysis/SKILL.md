# 数据分析技能  

## 支持的输入数据类型  

1. 表格形式的数据：csv、xlsx  
2. 文本中包含的数据：docx、pdf、md  
3. 图片中包含的数据：png、jpg  

## 步骤一：数据读入与探查  

**严禁**直接打印全量数据或将其加载到 Prompt 上下文中，只允许将必要的数据概要加载进上下文。  

### 1. 数据读入与保存  

- csv: 直接使用pandas库读取
- xlsx: 使用pandas库或 \xlsx 工具读取
- docx: 使用 \docx 工具读取，提取目标数据保存为csv
- pdf: 使用 \pdf 工具读取，提取目标数据保存为csv
- md: 直接读取，提取目标数据保存为csv
- png、jpg: 使用视觉工具读取，提取目标数据保存为csv

### 2. 数据探查  

**原则：** 只看样貌，不看全貌。

**标准探查代码模式:**
```python
import pandas as pd

file_path = 'target_data.csv'
try:
    df_preview = pd.read_csv(file_path, nrows=5)
    print("=== 数据预览 (前5行) ===")
    print(df_preview.to_markdown(index=False))
    print("\n=== 列信息与类型 ===")
    print(df_preview.info())
except Exception as e:
    print(f"摄入错误: {e}")
```

### 3. 编码与格式自适应  

- 编码回退: 尝试 `encoding='gbk'` 或 `encoding='latin1'`
- 分隔符: 尝试 `sep=';'` 或 `sep='\t'`

## 步骤二：数据质量检查  

### 1. 完整性  
- 检查 NULL/NaN/空字符串比例
- 检查记录数量是否符合预期
- 检查时间序列是否连续

### 2. 唯一性  
- 检查完全重复行
- 检查主键唯一性
- 检查业务实体重复（如大小写不同）

### 3. 准确性与有效性  
- 数值字段是否混入字符串
- 日期字段格式是否正确
- 取值范围是否合理

### 4. 一致性  
- 单位是否统一
- 枚举值是否一致
- 跨表数据是否一致

### 5. 关联性  
- 外键约束检查（孤儿数据）

## 步骤三：确定工作流程  

### 核心分析方法  

#### 1. 统计分析  
- 描述性统计：均值、中位数、标准差
- 推断性统计：假设检验、置信区间
- 相关性分析：皮尔逊、斯皮尔曼相关系数

#### 2. 模式发现  
- 趋势分析与时间序列分解
- 聚类：K-means、层次聚类、DBSCAN
- 关联规则挖掘

#### 3. 商业智能  
- KPI 分析与仪表板设计
- 客户细分与画像
- A/B 测试与 ROI 分析

### 核心可视化方法  

- 数值型：直方图、箱线图、散点图、折线图
- 分类型：条形图、饼图、热力图
- 时间序列：折线图、面积图、季节性分解

## 步骤四：规范代码生成与执行  

### 常用技术栈（Python）  
- 核心处理: pandas, numpy
- 统计与挖掘: scipy, sklearn
- 可视化: matplotlib, seaborn, plotly

### 代码生成步骤  

1. **需求分析**：明确目标，选择合适的库
2. **结构设计**：模块化代码，分解为小函数
3. **实施**：编写简洁高效的代码，加入错误处理

### 错误处理  

- 数据加载错误：文件丢失、格式问题、编码问题
- 内存问题：分块处理或数据采样
- 性能问题：优化算法，使用缓存和并行处理

## 步骤五：结果验证与输出

- [ ] 关键计算结果是否非空且数值合理？
- [ ] 输出文件是否包含数据且表头正确？

### 格式一致性：  
- [ ] 图表是否有标题、轴标签和图例？
- [ ] 报告字体是否统一？
